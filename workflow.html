Park_Sense — System Workflow

Visual overview of how the system works — CLI routing, code flow, and future phases.

1. CLI Routing — How main.py decides what to do

Depending on the flags you pass, main.py routes to either the full video loop or single-frame mode. Config is loaded first, then CLI flags override any values before anything runs. 

flowchart TD
    CLI(["$ python main.py ..."]):::io

    CLI --> LC["load_config()\nutils/config.py"]
    LC --> AO["apply_overrides()\n--confidence / --model"]
    AO --> GV["get_video_path()\n--video or config default"]

    GV --> ROUTE{"--frame\nprovided?"}

    ROUTE -- yes --> RF["run_frame()\ncore/pipeline.py"]
    ROUTE -- no  --> RV["run()\ncore/pipeline.py"]

    RF --> FSHOW{"--no-display?"}
    FSHOW -- no  --> FDISP["cv2.imshow\nshow frame window"]
    FSHOW -- yes --> SKIP["skip window"]

    RF --> FSAVE{"--save\nprovided?"}
    FSAVE -- yes --> FIMG["cv2.imwrite\nsave image"]
    FSAVE -- no  --> NOTHING["display only"]

    RV --> VSAVE{"--save\nprovided?"}
    VSAVE -- yes --> VVID["VideoWriter\nsave video"]
    VSAVE -- no  --> VDISP["display only"]

    classDef io fill:#1e3a5f,stroke:#7ec8e3,color:#fff

3. Code-Level — Single Frame Mode (--frame N)

When --frame N is passed, the pipeline jumps directly to that frame, runs detection once, then shows and/or saves it. 

flowchart LR
    CLI(["--frame 10"]):::io --> EF

    subgraph VIDEO["utils/video.py"]
      EF["extract_frame()\nseek to frame N\nvalidate range"]
    end

    EF --> RS["cv2.resize()"]

    subgraph DET["core/detector.py"]
      RS --> INF["YOLOv8 inference"]
      INF --> OUT["return detections"]
    end

    OUT --> subgraph VIZ["utils/visualizer.py"]
      DR["draw_detections()"]
    end

    DR --> LABEL["draw frame label\n'Frame N — X vehicle(s)'"]

    LABEL --> S1{"--save?"}
    S1 -- yes --> IMG["cv2.imwrite()\nsave to outputs/"]
    S1 -- no  --> ND

    LABEL --> ND{"--no-display?"}
    ND -- no  --> SHOW["cv2.imshow()\nwaitKey(0)"]
    ND -- yes --> DONE(["done"]):::io
    SHOW --> DONE

6. CLI Reference

All flags are optional. Config values are defaults — CLI flags override them.

FlagTypeDescription
--videopathVideo file to process
--configpathConfig file (default: config.yaml)
--confidencefloatDetection threshold, e.g. 0.6
--modelpathYOLO model to use, e.g. models/yolov8m.pt
--frameintExtract and analyze a single frame by number
--savepathSave output — image if --frame used, video otherwise
--no-displayflagSkip opening a window (useful when only saving)
